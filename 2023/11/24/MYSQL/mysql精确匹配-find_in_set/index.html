mysql精确匹配-find_in_set

引言

有时我们在做数据库查询时，需要得到某字段中包含某个值的记录，但是它也不是用like能解决的，使用like可能查到我们不想要的记录，它比like更精准，这时候就要用到mysql的FIND_IN_SET函数了。

步骤

  1. 例如，创建cc表，list字段，需要查询出list包含11的数据：

  2. 没有用过find_in_set的同志们，一般来说会想到用like来模糊查询这样的数据，例如：

  3. 这样查询虽然可以查到11的数据，但也会查询到不要的数据。：

  4. 使用find_in_set查询,可以精确的查询到包含11的数据：

  1. find_in_set的使用方法：

    find_in_set(‘你要查询的字符串’,’你的字段名称’)

  2. 注意事项：

    • 使用find_in_set会全表扫描。
    • 查询的字段数据用英文逗号拼接的。
    • 在数据量多的情况下不合适使用，会全表扫描，不会使用索引。
