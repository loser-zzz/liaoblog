array_merge和array+array的区别

引言

想起最近水群，看到的一个问题，就是array_merge函数和array+array的合并方式有什么区别。

总结

  • 相同之处：可以合并数组。
  • 不同之处:
    • array_merge相同字符串键名的情况下，后一个值会覆盖前一个值。在数值键名的情况下，会返回一个新数组，从o开始的新索引。
    • array+array键名相同（字符串和数值）的情况下，会保留前一个值，放弃后一个值。
